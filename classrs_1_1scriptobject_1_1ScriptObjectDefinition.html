Detailed Description

An abstract type which describes the object definition (metadata) of a scriptable object.

The metadata includes the number of fields, their names and their types.

Member Function Documentation

void rs::scriptobject::ScriptObjectDefinition::CalculateHash ( ScriptObjectHash  digest) const

Calculates a digest (hash) based on the field names and types of the object definition.

The hash can be compared with other hashes to determine equality (i.e. the compared object instances have the same object definition).
