A enumeration which controls the behaviour of the Merge method.

Enumerator

the default strategy, compares each field in turn in the source objects

Back 

the rhs object fields are placed at the front of the new object after merge

static unsigned rs::scriptobject::ScriptObject::CalculateSizeOverhead ( unsigned  fieldCount)
static

Calculates the size overhead of the 'management' data of the script object.

The overhead is the data portion of the script object which is not the actual field data itself. For example field offsets, indexes and key references.

Parameters
fieldCountthe number of fields to use in the calculation
Returns
The overhead size in bytes
